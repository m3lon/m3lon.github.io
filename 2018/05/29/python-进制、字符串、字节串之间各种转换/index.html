进制转换

(以下默认为python环境 16代表16进制)：

  • 十六进制转十进制: int(“0x10”,16) 或者直接输入0x16回车

  • 十进制转十六进制: hex(16)

    int(x, base=10) 函数用于将一个字符串或数字转换为整型。

    oct(x) 函数将一个整数转换成8进制字符串。

    bin(x) 返回一个整数 int 或者长整数 long int 的二进制表示。
